# harshit.ai — Portfolio Source

Source code for [harshit.ai](https://harshit.ai), the personal portfolio of Harshit Sharma — AI Product Manager.

## Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Deployment:** Vercel

## Features

- Project case studies with content type and domain tag filtering
- Open Graph meta tags for link previews
- Google Search Console integration
- Professional email via Google Workspace
